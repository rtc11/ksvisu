Show stateful processor nodes:
> ksvisu topology.desc

Show all nodes:
> ksvisu topology.desc -a

Diff between old and new topology description:
> ksvisu old.desc -d new.desc

Diff between old and new stateful processor nodes:
> ksvisu old.desc -d new.desc

Diff between all old and new nodes:
> ksvisu old.desc -d new.desc -a

Open GUI:
> ksvisu topology.desc -g
